# Детектор сундуков Minecraft на базе YOLO26

Этот репозиторий содержит набор скриптов, позволяющих:

1. **Собрать датасет** из ~500 изображений сундуков Minecraft, автоматически скачанных из интернета.
2. **Обучить модель** YOLO26 на собранном датасете.
3. **Запустить детектор** в режиме реального времени с визуализацией результатов в окне `Tkinter`.

> ⚠️ **Внимание:** для работы скриптов требуется доступ к интернету и возможность устанавливать Python‑библиотеки через `pip`.

## Структура проекта

| Файл | Назначение |
| --- | --- |
| `download_dataset.py` | Скачивает изображения сундуков по запросу и создаёт метки. |
| `chest.yaml` | Описание датасета для обучения YOLO. |
| `train.py` | Запуск обучения модели YOLO26. |
| `detect_gui.py` | Детектор сундуков с выводом в окне Tkinter. |
| `clone_repo.sh` | Bash‑скрипт, клонирующий этот репозиторий. |
| `clone_repo.bat` | Batch‑скрипт (Windows) для клонирования репозитория. |

## Установка

1. Установите Python 3.10+.
2. Установите зависимости:

   ```bash
   pip install ultralytics duckduckgo_search pillow pyautogui
   ```

## Использование

1. **Скачайте датасет:**

   ```bash
   python download_dataset.py
   ```

   Скрипт скачает изображения по запросу `minecraft chest different angles` и создаст папку `data/images/train` с картинками и папку `data/labels/train` с метками.

2. **Обучите модель:**

   ```bash
   python train.py
   ```

   Будет использована базовая модель `yolo26n.pt`. После обучения веса будут сохранены в `runs/chest/train/weights/best.pt`.

3. **Запустите детектор:**

   ```bash
   python detect_gui.py
   ```

   Откроется окно, в котором каждые 1 секунду будет захватываться экран и отображаться найденные сундуки с красными рамками.

## Скрипты для клонирования

`clone_repo.sh` и `clone_repo.bat` содержат команды для скачивания этого репозитория через `git`. Вы можете запустить соответствующий скрипт на Linux/macOS или Windows, чтобы клонировать проект:

```bash
./clone_repo.sh
```

или в Windows:

```bat
clone_repo.bat
```

Они автоматически выполняют команду `git clone`.
